### 숫자만 추출

---
설명 <br>
문자와 숫자가 섞여있는 문자열이 주어지면 그 중 숫자만 추출하여 그 순서대로 자연수를 맍듭니다.
만약 "g0en2T0s8eSoft"에서 숫자만 추출하면 2, 0, 8 이고 이것을 자연수를 만들면 208 이다.
추출하면 만들어지는 자연수는 100,000,000을 넘지 않습니다.

- 입력

첫 줄에 숫자가 썩인 문자열이 주어집니다. 문자열의 길이는 100을 넘지 않습니다.

- 출력

첫 줄에 자연수를 출력합니다.

```
예시 입력

g0en2T0s8eSoft
```

```
예시 출력

208
```
